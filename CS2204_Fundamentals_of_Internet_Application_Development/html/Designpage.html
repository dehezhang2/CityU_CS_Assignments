Free design

(m)

(1)"Positioning scheme absolute" is used in header, footer style and blocks of homepage and orderpage.
(2)"Float" is used in the menu items .

(n)CSS3 techniques:
(1)Round corner and border-radius is used in blocks of homepage and orderpage.
(2)Box-shadow is used in blocks of homepage.
(3)Animation is used in menu items and order block of orderpage.
(4)Transform:rotate is used in menu items.

(o)Style are grouped only in style sheets.
(1)Style are grouped in a sequence of content.
(2)Type selector, ID selector ,class selector,group selector are used in the style sheets.
(3)Child selector is used in the style sheets.
(4)Attribute selector , pseudo class and pseudo element are used in the style sheets.

Other:

(1)Background-image is used in all pages.

(2)All headers have a same background image and all headers and footers are in a same style(font style and background).

(3)Hover is widely used

  • The menu items are displayed by hover,and combine with animation of opacity.
  • The blocks in homepage are designed by hover,when pointed by mouse the opacity and shadow will change so it looks more solid.
  • The "shop information" block was divide into 5 parts which were displayed as the same method menu items used.
  • The "tr" of table in order page uses hover to change its color.

Javascript eventhandler

1.In HTML

  1. Navigastion(free design) in homepage and orderpage.
  2. Navigation(free design) of shop information in homepage.
  3. Init function to show navigation menu, shop information, promotion information and video when page loaded in homepage.
  4. Onended of the video to cycle the video in homepage.
  5. Onsubmit of the form in homepage.
  6. Init function to show navigation menu, menu items of sushi, reset recal() function and select some object after page loaded.
  7. Onclick of three heading of menu.

2.In JS

  1. Add event click to "undo".
  2. Add event click to each button of order items.
